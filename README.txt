To-Do List Manager
-------------------

This is a simple Python-based To-Do List Manager application that allows users to:
1. View tasks
2. Add tasks
3. Mark tasks as complete
4. Delete tasks

**How to Run the Application:**

1. Using Python:
   - Clone the repository from GitHub.
   - Run the application using the command: `python To_Do_List.py`.

2. Using Docker:
   - Pull the Docker image from Docker Hub: `docker pull berk0039/assignment4`.
   - Run the application using the command: `docker run -it berk0039/assignment4`.
